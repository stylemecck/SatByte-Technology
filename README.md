# SatByte Technology

**High-End Web Solutions | Custom Portals | Automated Business Architectures**

SatByte Technology is a premium web development agency platform engineered for scalable agency growth, secure client portals, high-conversion project quoting, and enterprise-grade content management.

## 🚀 Recent Modernization (Phase V2)

We have recently completed an extensive UI/UX overhaul across the entire platform, introducing high-end design patterns and performance optimizations.

### 🍱 Bento-Style Architecture
- **Grid-Based Efficiency**: Transitioned project cards, service grids, and dashboard stats to the ultra-modern "Bento" layout.
- **Visual Rhythm**: Standardized on negative space, tracking, and premium contrast to create a world-class aesthetic.

### 🌗 Adaptive Theme System
- **Semantic Token Engine**: Implemented a global CSS variable system (`--background`, `--foreground`, `--primary`, etc.) for seamless theme switching.
- **Persistence**: User theme preferences (Light/Dark) are now persisted across sessions and subdomains.

### 🏝 Modern Navigation
- **Floating Island Navbar**: A dynamic, centered "Pill" design that transitions smoothly on scroll, providing an immersive interaction model.
- **Categorized Hubs**: Re-engineered footers and nav-menus for better logic and accessibility.

### 📱 Responsive Support Excellence
- **Mobile Ticket Flow**: Optimized the support ticketing system for mobile viewports, including full-screen creation modes and intuitive back-navigation.

## 🚀 Core Features

### 💎 Premium Portals
- **High-Fidelity Minimalism**: Standardized on **Inter** typography.
- **Glassmorphism 2.0**: Enhanced dashboard interfaces using advanced backdrop filters.
- **Fluid Micro-Animations**: Refined Framer Motion interactions.

### 🔐 Advanced Authentication
- **Hybrid Login Flow**: Frictionless OTP entry or permanent Secure Password login.
- **Cross-Domain Persistence**: Shared authentication state via secure cookie relay.

### 💼 Talent & Career Infrastructure
- **Integrated Admin CMS**: Full CRUD control for Jobs, Internships, and Certifications.
- **Candidate Pipeline**: Real-time review of applications and status management.

### 💳 Financial Infrastructure
- **Triple-Gateway Integration**: Native support for **Stripe**, **Razorpay**, and **PayUMoney**.
- **Branded Invoicing**: Automatic PDF generation and SB-Reference tracking.

## 🛠 Tech Stack

- **Frontend**: React 19, TypeScript, Vite, Tailwind CSS, Framer Motion, Radix UI.
- **Backend API**: Node.js, Express, JWT, Bcrypt.
- **Database**: MongoDB Atlas (Mongoose ORM).
- **Core Cloud**: Cloudinary (Assets), Stripe/Razorpay/PayU (Payments).
- **Automation**: Nodemailer (Transactional Email), PDFKit (Invoices).
